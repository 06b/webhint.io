## Getting started

Clone the project:

``` bash
$ git clone https://github.com/MicrosoftEdge/Sonar-Website.git
```

Install dependencies:

```
$ npm install
```

Build the site:

```bash
$ npm run build
```

Run server:

```bash
$ npm start
```
